# WeChat-2048

一个完整的微信小程序版本 2048 游戏。

## 功能特性

✨ **完整的 2048 游戏逻辑**
- 4x4 棋盘
- 滑动控制（上下左右）
- 自动合并相同数字
- 随机生成新方块

📊 **游戏统计**
- 实时得分显示
- 最高分记录（本地存储）
- 游戏结束检测
- 获胜提示（达到 2048）

🎮 **交互体验**
- 流畅的触摸滑动操作
- 清晰的视觉反馈
- 响应式界面设计
- 支持重新开始游戏

## 游戏规则

1. **目标**：通过合并方块，达到 2048
2. **操作**：用手指在屏幕上上下左右滑动来移动方块
3. **合并**：两个相同的数字相碰撞会合并成一个，数值为两个数字之和
4. **新方块**：每次移动后，会在随机空位生成一个新方块（90% 概率为 2，10% 概率为 4）
5. **游戏结束**：当棋盘满了且无法进行合并时游戏结束

## 开发指南

### 环境要求
- 微信开发者工具
- 有效的微信小程序 AppID

### 本地运行

1. 下载或克隆本仓库
2. 打开微信开发者工具
3. 选择「导入项目」
4. 选择项目目录
5. 输入 AppID（测试号可使用）
6. 点击「导入」即可开始开发

### 主要文件说明

#### `pages/index/index.js`
- `initGame()`：初始化游戏
- `move(direction)`：处理移动逻辑
- `mergeLine(line)`：合并一行中的相同数字
- `isGameOver(board)`：检查游戏是否结束
- `hasWon(board)`：检查是否赢得游戏

#### `pages/index/index.wxml`
- 显示得分和最高分
- 渲染 4x4 棋盘
- 显示所有方块
- 提供重新开始按钮

#### `pages/index/index.wxss`
- 棋盘和方块样式
- 颜色主题设计
- 响应式布局

## 常见问题

**Q: 如何修改棋盘大小？**
A: 修改 `pages/index/index.js` 中的 `boardSize` 属性和对应的 WXML 中的循环次数。

**Q: 如何改变颜色主题？**
A: 编辑 `pages/index/index.wxss` 中的颜色值，或修改 `getTileColor()` 函数中的颜色映射。

**Q: 如何调整游戏速度？**
A: 游戏没有速度限制，可在 WXSS 中修改 `transition` 属性的时间值。
